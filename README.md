The data used in this excel project is BigMart Sales Data. This project is all about
1.Loading the data into excel
2.Cleaning the data i,e. Fill the null values with a value if present, Check if the values are unique or not, Checking the values are consistent or not
3.Data preprocessing i,e. Relationships between the attributes and creating new attributes from existing ones.
4.Data Analysis--> This can be done only by creating pivot tables and Analysing the data.
5.Creating stunning dashboards to understand the data and the relationships among the attributes.
